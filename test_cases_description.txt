Test Case 1: Login

Objective:
    Verify that user can log in with valid credentials.

Preconditions:
    The user has access to the website https://www.saucedemo.com.
Steps:
    1. Open browser and navigate to the website.
    2. Enter username "standard_user".
    3. Enter password "secret_sauce".
    4. Click the "Login" button.
Expected Result:
    The inventory page is displayed with the list of products.


Test Case 2: Verify Product List

Objctive:
    Verify that the list of products is visible after login.

Steps:
    1. Log in using valid credentials.
    2. Wait for the product list to load.
    3. Check that product names are visible on the page.
Expected Result:
    At least one product item is displayed.


Test Case 3:
Add To Cart

Objective:

Steps:
